# Item 4 – Transformações e Preparação dos Dados (Silver)

Nesta etapa, os dados da camada Raw foram tratados e padronizados, resultando em uma base mais consistente e adequada para análises posteriores.

As principais transformações realizadas incluem:
- seleção dos campos relevantes para o caso de uso
- remoção de registros duplicados
- tratamento de valores ausentes
- conversão de tipos de dados
- padronização de textos e categorias

O resultado foi armazenado na camada Silver, representando dados limpos e organizados, prontos para consumo analítico e aplicações de inteligência artificial.
